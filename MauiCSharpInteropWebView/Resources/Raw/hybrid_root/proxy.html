Requests can be proxied through .NET to allow direct access to native APIs, and allow custom response streams to be returned. Request URLs can be created with the format of "https://0.0.0.0/proxy?" followed by custom query string parameters.
Round trip .NET invoke

This button will invoke a .NET method and return the result to the JavaScript code. This is useful for offloading work to .NET and returning the result to the JavaScript code..



Create proxy URLs and pass into native HTML elements

The following buttons will create a proxy URL and pass it into an image element. The image request will will be handled by a custom .NET method.



Interactive map with offline tiles.

The map tiles are locally stored in a MBTiles file (a SQLite database file with a specific table schema and .mbtiles extension). The MBTile file used in this example contains 4,096 map tiles. storing this as an MBTiles/SQLite file makes it significantly easier to manage and distribute. The map is displayed using MapLibre GL JS.
